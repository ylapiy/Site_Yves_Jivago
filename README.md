# 🌐 Site Institucional – Yves Jivago

Este é um projeto de site institucional desenvolvido para apresentar os serviços jurídicos de **Yves Jivago**, com foco em uma advocacia moderna, estratégica e acessível.

---

## 📌 Sobre o Projeto

O objetivo deste site é fornecer uma presença digital profissional, permitindo que clientes conheçam melhor o advogado, suas áreas de atuação e formas de contato.

A aplicação possui uma interface simples, elegante e de fácil navegação.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts

---

## 📂 Estrutura do Projeto

```
site-do-yves/
│── index.html
│── style.css
│── Yves.jpeg
```

---

## 🎯 Funcionalidades

* Navegação por seções (scroll suave)
* Página institucional com:

  * Início
  * Sobre
  * Áreas de atuação
  * Contato
* Layout organizado e intuitivo
* Botão de chamada para ação (CTA)

---

## 💻 Como Executar o Projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a pasta do projeto:

```
cd site-do-yves
```

3. Abra o arquivo `index.html` no navegador.
